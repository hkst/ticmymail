## If Copilot throws git/path errors (Windows/PowerShell)

*   **Branch exists:**  
    `git switch chore/flatten-src-tmm` (no `-c`).  
    Create a new one: `git switch -c ft_bootstrap_part2`.
*   **git mv ‘source directory is empty’:**  
    Create the missing path first (e.g., `src/tmm/tests/.keep`) or use `New-Item -ItemType Directory src/tmm/tests` and then add files—`git mv` only works on tracked files/dirs.
